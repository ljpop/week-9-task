Trello Clone (Drag’n’Drop API)

Mozete koristiti neku UI biblioteku/framework (Bootstrap, MaterialUI, Tailwind, itd.)

Zahtevi:
- kreirati Trello Clone aplikaciju koristeci Drag'nDrop API
- 4 kolone (npr. Backlog, In Progress, Complete, On Hold)
- Add Item funkcionalnost unutar svake kolone
- Reorder elemenata unutar kolone i izmedju razlicitih kolona
- Sacuvati izmene i kada se refreshuje stranica
